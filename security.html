Security Overview

My Private Notes for Jira is built with security as a foundational principle. We leverage Atlassian's enterprise-grade Forge platform to ensure your private notes remain private.

Architecture Security

Forge Platform

Our app is built on Atlassian Forge, a serverless platform that runs entirely within Atlassian's infrastructure. This means:

  • No external servers ‚Äî All code execution happens within Atlassian's secure environment.
  • No data egress ‚Äî Your notes never leave Atlassian's infrastructure.
  • Automatic updates ‚Äî Security patches are applied automatically by Atlassian.

Data Storage

All data is stored using Forge Storage API:

  • Encrypted at rest ‚Äî All stored data is encrypted using industry-standard encryption.
  • Encrypted in transit ‚Äî All data transmission uses TLS encryption.
  • User-scoped storage ‚Äî Each user's notes are isolated and cannot be accessed by other users.
