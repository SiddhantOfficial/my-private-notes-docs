Architecture Overview

My Private Notes is built on Atlassian Forge, a secure-by-design platform for Atlassian apps:

  • Forge Runtime: The app runs entirely within Atlassian's sandboxed environment.
  • Forge Storage API: Notes are stored using Atlassian's managed key-value storage.
  • No Egress: The app does not make any external network requests.
  • User Isolation: Storage keys are prefixed with user account IDs, ensuring complete data isolation.

Data Access Controls

We implement strict access controls to protect your data:

  • User-Bound Storage: Notes are keyed using accountId:issueKey, making them accessible only by the creating user.
  • No Admin Access: Jira administrators cannot access individual users' private notes.
  • No Developer Access: Even we (the app developers) cannot access your stored notes.
